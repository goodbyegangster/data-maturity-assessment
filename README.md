# data-maturity-assessment

データ成熟度評価サービス。

![demo](https://github.com/user-attachments/assets/e724d880-c0e6-4652-a076-488b16b3af73)

## コンテナ起動

```shell
docker run --rm --name data-maturity-assessment -p 8080:8080 ghcr.io/goodbyegangster/data-maturity-assessment:latest
```

### 出典資料 (Source Materials)

本サービスで使用している評価項目データは、以下の資料に基づいています。

1. 独立行政法人情報処理推進機構 (IPA)
   - 資料名: [データマチュリティアセスメント 解説書](https://www.ipa.go.jp/digital/data/data-spaces-academy.html)
   - ライセンス: [CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.ja)

上記資料は、IPA が以下の政府資料を翻訳・加工したものです。

1. 英国政府 (UK Government)
   - 資料名: [Data maturity assessment for government: framework](https://www.gov.uk/government/publications/data-maturity-assessment-for-government-framework)
   - ライセンス: [Open Government Licence v3.0](https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/) (CC BY 4.0 互換)
2. 豪州政府 (Australian Government)
   - 資料名: [Data Maturity Assessment Tool](https://www.finance.gov.au/government/public-data/public-data-policy/data-maturity-assessment-tool)
   - ライセンス: [Creative Commons Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/)
